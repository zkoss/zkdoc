**Property:**

`org.zkoss.zul.listbox.groupSelect`

`Default: false`  
`[Since 5.0.7]`

Giving true on this property makes Listgroups selectable in Listbox.

``` xml
<listbox>
    <custom-attributes org.zkoss.zul.listbox.groupSelect="true"/>
    ...
</listbox>
```
