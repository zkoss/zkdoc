**Property:** org.zkoss.theme.preferred

{% include global-scope-only.html %}

Default: `none`

If not specified, zk will load the default theme. ZK has different default theme before and after 8.5, see [ZK Official Themes](/zk_dev_ref/theming_and_styling/zk_official_themes). 
This property allows you to specify the preferred theme you want to load, it will override the default theme.
