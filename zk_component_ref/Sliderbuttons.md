# Employment/Purpose

A pair of draggable buttons defining the start value and the end value
in a Multislider.

# Example

![](images/ZKComRef_Sliderbuttons.png)

``` xml
<zk>
  <multislider>
    <sliderbuttons startValue="10" endValue="70"/>
  </multislider>
</zk>
```

# Properties

## StartValue, EndValue

Represent the range value. (Default: 0)
