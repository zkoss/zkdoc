## Keyboard Support

| Key | Description |
|---|---|
| Alt + ArrowUp / Alt + ArrowDown | Open the date picker popup. |
| ArrowUp / ArrowDown / ArrowLeft / ArrowRight | (In popup) Navigate the date. |
| Enter / Spacebar | (In popup) Select the date. |
| Escape | (In popup) Close the popup. |
