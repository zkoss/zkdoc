## Keyboard Support

| Key | Description |
|---|---|
| ArrowUp / ArrowDown | Open the popup and navigate options. |
| ArrowLeft / ArrowRight | Navigate selections |
| Enter | Select the options |
| Escape | Close the popup |
| Delete / Backspace | Remove the focused selection. |
