## Keyboard Support

(Only available if <b>linear</b> is `false`)

| Key | Description |
|---|---|
| ArrowLeft / ArrowUp | Focus on the previous step. |
| ArrowRight/ ArrowDown | Focus on the next step. |
| Enter / Spacebar | Select the step. |
