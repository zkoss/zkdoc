## Keyboard Support

| Key | Description |
|---|---|
| ArrowUp / ArrowLeft / ArrowRight | Navigate options. |
| ArrowDown | Open the popup or navigate options |
| Enter / Spacebar | Select the options |
| Escape | Close the popup |
| Backspace / Delete | Clear selection |
