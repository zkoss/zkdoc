## Keyboard Support

| Key | Description |
|---|---|
| <b>Horizontal:</b> ArrowLeft / ArrowRight<br/>
<b>Vertical:</b> ArrowUp / ArrowDown | Navigate tabs. |
| Enter / Spacebar | Select the tab. |
| Delete | Close the tab. (only if <b>closable</b> is enabled) |
