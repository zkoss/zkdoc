## Keyboard Support

| Key | Description |
|---|---|
| ArrowUp / ArrowRight | Increases the rating value. |
| ArrowDown / ArrowLeft | Decreases the rating value. |
| Home | Sets the rating value to 0. |
| End | Sets the rating to its maximum value. |
