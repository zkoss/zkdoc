- Style Guide: [

# Employment/Purpose

A `tabpanel`is the body of a single tab panel. You would place the
content for a group of components within a tab panel. The first
`tabpanel`corresponds to the first `tab,`the second
`tabpanel`corresponds to the second `tab`and so on.

# Example

![](images/ZKComRef_Containers_Tabs.PNG)

``` xml
<zk>
    <tabbox width="400px">
        <tabs>
            <tab label="Tab 1" />
            <tab label="Tab 2" />
        </tabs>
        <tabpanels>
            <tabpanel>This is panel 1</tabpanel>
            <tabpanel>This is panel 2</tabpanel>
        </tabpanels>
    </tabbox>
    <space />
    <tabbox width="400px" mold="accordion">
        <tabs>
            <tab label="Tab 3" />
            <tab label="Tab 4" />
        </tabs>
        <tabpanels>
            <tabpanel>This is panel 3</tabpanel>
            <tabpanel>This is panel 4</tabpanel>
        </tabpanels>
    </tabbox>
</zk>
```
