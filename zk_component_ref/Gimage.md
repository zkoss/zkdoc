# Gimage

- Demonstration:
  [Gmaps](http://www.zkoss.org/zkdemo/reporting/google_map)
- Java API: N/A
- JavaScript API: N/A
- Style Guide: N/A

# Employment/Purpose

Image anchored on two specified (latitude, longitude) points(south-west
and north-east) which is zoomed and moved with the Google Maps.
