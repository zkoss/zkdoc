

# Gmarker

- Demonstration:
  [Gmaps](http://www.zkoss.org/zkdemo/reporting/google_map)
- Java API: N/A
- JavaScript API: N/A


# Employment/Purpose

The gmarker of the Google Maps that you can used to indicate a specific
(latitude, longitude) points. The icon, shadow, color, and so on are
customizable.
