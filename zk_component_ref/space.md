# Employment/Purpose

Space is a Separator with the orient default to "horizontal". In other
words, <space> is equivalent to <separator orient="horizontal">

# Example

![](/zk_component_ref/images/ZKComRef_Space_Example.png)

```xml
 <zk>
     <space bar="true"/>
     <space bar="true"/>
     <space bar="true"/>
     <space bar="true"/>
 </zk>
```
