The complete, runnable source for the snippets in this book lives in
the [zkchartsessentials](https://github.com/zkoss/zkchartsessentials)
GitHub repository. Clone it and run with Maven:

```sh
git clone https://github.com/zkoss/zkchartsessentials.git
cd zkchartsessentials
mvn jetty:run
```

Then open <http://localhost:8080/> to browse the examples.
